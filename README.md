# tictactoe.clj

## Description

This is an implementation of the tic-tac-toe game (or Noughts and crosses, Xs and Os) in Clojure.

## Dependencies

##### Execution
* Clojure 1.7

##### Testing
* [Speclj][speclj] 3.3.1

[speclj]: https://github.com/slagyr/speclj

##### Others
* [Leiningen][lein] (for managing the project)

[lein]: http://leiningen.org/

## Setup

##### Run tests
`lein test`
